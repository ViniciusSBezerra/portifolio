Pokedex

Este Projeto esta sendo realizado com o proposito de fazer uma pokedex com todos os pokemons até o momento.
O projeto está em andamento, no momento a única funcionabilidade é consumir a api da Pokeapi, e mostrar todos os 898 pokemons existentes até o momento ou todos os pokemons disponíveis na ultima versão da api.
As funcionabilidades planejadas até o momento são, fazer uma buscas por nome ou id do pokemon, buscar pelo elemento do pokemon e criar um modal para quando o usuário clicar em algum pokemon mostrar mais informações do mesmo.
Projeto sendo realizado com React.js, Axios e Styled-Components.
