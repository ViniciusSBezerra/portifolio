Tecnologias

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. É mantido pelo Facebook, Instagram, outras empresas e uma comunidade de desenvolvedores individuais.

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript.

Utilizando literais de modelo com tags (uma adição recente ao JavaScript) e o poder do CSS, styled-components permite que você escreva código CSS real para estilizar seus componentes

Pokedex

Este Projeto esta sendo realizado com o proposito de fazer uma pokedex com todos os pokemons até o momento. O projeto está em andamento, no momento a única funcionabilidade é consumir a api da Pokeapi, e mostrar todos os 898 pokemons existentes até o momento ou todos os pokemons disponíveis na ultima versão da api. As funcionabilidades planejadas até o momento são, fazer uma buscas por nome ou id do pokemon, buscar pelo elemento do pokemon e criar um modal para quando o usuário clicar em algum pokemon mostrar mais informações do mesmo
Projeto sendo realizado com React.js, Axios e Styled-Components.
